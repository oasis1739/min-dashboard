# 원가분석 대시보드

이앤비정보통신(민두식) · 엠제이(MJ) 케이블 하도급 — 2025년 원가분석 대시보드입니다.

단일 HTML 파일로 동작하는 정적 웹페이지이며, 모바일·PC 어디서나 브라우저로 열람할 수 있습니다.
데이터는 구글시트와 실시간으로 연결되어 표시됩니다.

## 구성

- `index.html` — 대시보드 본체 (이 파일 하나로 동작)

## 사용

GitHub Pages 로 호스팅하여 링크로 접속합니다.
접속 시 비밀번호 입력이 필요합니다.

## 특징

- 한 페이지 스크롤 + 상단 고정 메뉴(연간요약 / 월별손익 / 매출분석 / 비용분석)
- 입력값은 구글시트에 자동 저장되고, 시트 수정 내용도 자동 반영(양방향 동기화)
- 차트 시각화 내장
